## When to Contact Support

### Immediate Contact Recommended
- Complete loss of internet connectivity
- Speeds consistently below 50% of plan speed
- Router or equipment showing error lights
- Suspected fiber line damage
- Security concerns (unauthorized network access)

### Can Wait for Self-Service
- WiFi password reset
- Basic speed test verification
- Device connection issues
- General questions about features
- Billing invoice questions

## Support During Installation

The 48-hour installation process includes comprehensive support:

1. **Pre-installation call**: Technical team contacts you to schedule and explain process
2. **Installation day**: Technician arrives within 2-hour window
3. **On-site setup**: Fiber installation, router configuration, WiFi setup
4. **Speed verification**: Full testing to ensure plan speeds delivered
5. **Tutorial**: Basic router and portal usage explanation
6. **24-hour follow-up**: Call to ensure everything working properly

<Info>
  Installation is handled by Telenet's own technical team ("Equipo técnico local") - not third-party contractors. This ensures quality and accountability.
</Info>

## Escalation Process

If your issue isn't resolved to satisfaction:

1. **Front-line support**: Initial contact through any channel
2. **Supervisor review**: Request escalation if unresolved after 30 minutes
3. **Technical specialist**: Advanced troubleshooting and diagnostics
4. **Engineering team**: Network infrastructure investigation
5. **Management review**: Final escalation for complex or unresolved issues

**VIP customers** can request supervisor-level support immediately.
